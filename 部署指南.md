# Google Ads Mastery Radar · 部署指南

## 你有两个文件
- `index.html` — 客户测验页面（发给客户的链接）
- `admin.html` — 你的后台管理页面（只有你能用）

---

## 第一步：注册 Supabase（免费数据库）

1. 去 https://supabase.com 注册账号（免费）
2. 点击 "New Project"，填写项目名称（比如 ads-radar），设置一个数据库密码
3. 等待项目创建完成（约1分钟）

---

## 第二步：创建数据表

1. 在 Supabase 左侧菜单点击 "SQL Editor"
2. 粘贴以下代码并点击 Run：

```sql
create table quiz_results (
  id uuid default gen_random_uuid() primary key,
  company text,
  email text,
  name text,
  score integer,
  total integer,
  percentage integer,
  level text,
  wrong_topics text[],
  correct_topics text[],
  completed_at timestamptz default now()
);

alter table quiz_results enable row level security;

create policy "Allow insert" on quiz_results for insert with check (true);
create policy "Allow select for service" on quiz_results for select using (true);
```

---

## 第三步：获取你的 API Keys

1. 在 Supabase 左侧菜单点击 "Project Settings" → "API"
2. 复制两个值：
   - **Project URL**（类似 https://xxxxx.supabase.co）
   - **anon public key**（很长的一串字符）

---

## 第四步：填入两个 HTML 文件

在 `index.html` 和 `admin.html` 中，找到以下两行并替换：

```javascript
const SUPABASE_URL = 'YOUR_SUPABASE_URL';      // 换成你的 Project URL
const SUPABASE_KEY = 'YOUR_SUPABASE_ANON_KEY'; // 换成你的 anon key
```

在 `admin.html` 中，还要设置你的管理员密码：

```javascript
const ADMIN_PASSWORD = 'YOUR_ADMIN_PASSWORD';  // 换成你想要的密码
```

---

## 第五步：部署到 Vercel（免费托管）

1. 去 https://vercel.com 注册账号（用 Google 账号即可）
2. 点击 "Add New Project" → "Upload"
3. 把 `index.html` 和 `admin.html` 这两个文件一起上传
4. 点击 Deploy

部署完成后你会得到两个链接：
- `https://your-project.vercel.app` → 发给客户的测验链接
- `https://your-project.vercel.app/admin` → 你的后台管理链接

---

## 后台使用说明

打开 admin.html 对应的链接，输入你设置的密码即可登录。

后台功能：
- 看到所有参与测验的客户公司、姓名、邮箱
- 查看每个客户的成绩和等级（A/B/C）
- 看到每个客户的薄弱环节（方便你针对性跟进）
- 按等级筛选，或搜索特定公司
- 点击"刷新数据"获取最新记录
